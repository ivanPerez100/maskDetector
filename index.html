Mask Detector

This is a Mask Detector that was created using Tensorflow. We loaded in the Mobilenet_v1 model from Tensorflow's Object Detection API and then used Transfer Learning to train this model on a Jupyter Notebook using annotated images from a kaggle dataset. This data set includes images of people either wearing, not wearing, or wearing a mask incorrectly from a kaggle dataset.

Once the model has completly loaded in, the "Enable Webcam" button would no longer be opaque and you would be able to start video.
